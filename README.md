Call Center Performance Dashboard

This Power BI dashboard provides an interactive view of call center performance, tracking key metrics such as Total Incoming Calls, Calls Resolved, Average Speed of Answer, and Answer Rate. It allows department-wise comparisons and insights using clean visuals and well-structured measures for accurate and actionable analysis.

Features:
Visualize call volume by department
Monitor answered vs unanswered calls
Track answer rates across departments
Interactive filters for months and departments
Clean, professional layout for easy interpretation
